## Excel Fitness Activity Analysis

### Objective
Analyze fitness tracker data to understand activity levels, calorie burn, and user engagement patterns.

### Dataset
Fitbit daily activity dataset.

### Tools
- Microsoft Excel
- Pivot Tables
- Excel formulas (IF, COUNTIF, AVERAGE)
- Data cleaning and validation

### Process
- Preserved raw data and documented transformations
- Cleaned dates and handled missing values using business logic
- Created KPIs and pivot tables
- Built a one-page dashboard for insights

### Key Insights
- Higher activity levels are associated with higher calorie burn
- Most days fall into low-to-medium activity levels

### Next Steps

- Build a Power BI dashboard
